## 📌 Notas

Esta versión usa **WASM-GC**, por lo que puede ser más rápida y estable que **JavaScript** en navegadores compatibles.

Si la pantalla aparece en blanco:
- Actualiza el navegador
- Espera unos minutos 
- O puede ser un error del archivo
